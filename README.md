# COUNTER

This project consists, as stated in the title, of a ***Counter***, a counter that can be used by any user to increase and/or decrease the value shown on the display.

In fact, the ***Counter*** is equipped with a display which shows the value and, underneath, two buttons corresponding to te - and +  which have the purpose of increasing or decreasing the value itself.

It was built using HTML5 and CSS3 for structure and design , implemented by JavaScript that allows the action of incrementing and decrementing the value (not below than zero).

The user just needs to click on the desired button  to increase or decrease the value as he pleases.

However, the value will be reset when the page is refreshed or reopened .

Round lines have been used to give a *soft* look to both, the Counter and the title, accompanied by shades of blue with some effect of shaded shadows of white color, maintaining the homogeneity of colors throughout the project.

All this  has been designed to make the use of the ***Counter*** simple and intuitive for any user and , at the same time, pleasing to the eye.
